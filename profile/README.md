We develop **machine learning methods for urban transportation and mobility**, with a focus on building models that are generative, probabilistic, and scalable. Our work spans deep generative modeling, spatiotemporal forecasting, reinforcement learning for connected vehicles, and large language/vision models for transportation safety.

## Open-Source Teaching Tools

We also build **AI Teaching Assistants (AITA)** — LLM-powered course chatbots deployed across multiple UMN engineering courses.

| Repo | Description |
|------|-------------|
